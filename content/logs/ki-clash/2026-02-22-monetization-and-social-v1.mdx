## What

This milestone brought monetization and social/retention features to Ki Clash across the web, server, and mobile clients.

### Ads (AdSense)

- Added a lobby banner and a between-match interstitial via reusable ad components (`AdBanner`, `InterstitialAd`) plus a `useAdTiming` hook (`web/src/components/ads/*`, `web/src/hooks/useAdTiming.ts`).
- Wired ads into the PvP page alongside pixel art (`web/src/app/pvp/page.tsx`).
- Injected the AdSense script tag into the root layout in an env-driven way so the publisher ID is not hardcoded (`web/src/app/layout.tsx`).
- Made ad rendering conditional so ads are suppressed for ad-free users, wired through `web/src/lib/api.ts`.

### Server: ranked + payments

One server commit (`9f10dc5`) shipped a ranked ELO system, a Stripe ad-free pass, and a leaderboard API together, backed by:

- An Alembic migration adding ranked and payment fields (`alembic/versions/b8f3c72e1a44_add_ranked_and_payment_fields.py`).
- A Stripe payment handler under `app/core/payment/` (`stripe_handler.py`).
- Dedicated ranked/payment/player schemas (`app/schemas/ranked.py`, `app/schemas/payment.py`, `app/schemas/player.py`) and services (`app/services/ranked_service.py`, `app/services/payment_service.py`).
- New endpoints (`app/api/v1/endpoints/ranked.py`, `purchases.py`, `players.py`) registered in `app/api/v1/router.py`.

Stripe was added as a project dependency in `pyproject.toml` (`4ad4628`).

### Web client surfaces

- Ranked leaderboard page (`web/src/app/ranked/page.tsx`).
- Ad-free shop page (`web/src/app/shop/page.tsx`).
- Match history page backed by the API (`web/src/app/history/page.tsx`).
- Friend challenge flow via share link with lobby navigation (`web/src/app/invite/page.tsx`, `web/src/app/page.tsx`).

### Mobile (Expo) parity

- Ranked leaderboard screen (`mobile/app/ranked.tsx`).
- Match history and friend invite screens (`mobile/app/history.tsx`, `mobile/app/invite.tsx`).
- Each wired through `mobile/src/lib/api.ts`.

## Why

Monetization v1 combines two revenue paths: ad impressions (AdSense lobby banner + interstitial) and a paid Stripe ad-free pass. Ranked ELO, a leaderboard, match history, and friend challenge were added together as retention surfaces, giving players reasons to return and to invite others. Mobile received parity screens so both clients expose the same features.
