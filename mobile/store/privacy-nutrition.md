# App Privacy Notes

Use this when filling App Store Connect privacy answers.

## Data Used To Operate The App
- User ID: guest player identifier and display name.
- Gameplay Content: match history, leaderboard records, room code lifecycle, submitted actions, and match results.
- Diagnostics: crash/error data, WebSocket failure events, API failure events, timestamps, user agent, and request metadata.
- Usage Data: page/screen style events, play starts, room creation, invite/share events, action submitted, and match finish events.

## Data Not Sold
JJAN! Ki Clash does not sell personal information.

## Payments
Card details are not stored by JJAN! Ki Clash. Platform, Lemon Squeezy, or Stripe payment processors handle payment details.

## Tracking
Do not mark cross-app tracking unless an ad SDK or tracking SDK is added to the native app build.
