## Description
JJAN! Ki Clash is a fast 1v1 reveal duel inspired by Korean ki-battle mind games.

Practice against AI, create a PvP room, send the code to a friend, and clash in real time. Each turn is a read: charge for more ki, block, strike, fire an energy wave, or teleport out of danger. Win the best-of-three match by timing your move better than your opponent.

Features:
- Fast AI practice matches
- Real-time PvP rooms with shareable codes
- Six original fighters
- Best-of-three ki battles
- Match history and leaderboard groundwork
- Mobile-ready haptics, sound, and secure guest sessions
